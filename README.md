# Remote Mouse Server

## Requirements
- Yarn

## Get started
- Set environment variables
    - Set a free port number for the websocket server in the `.env` file. Default is `4444`
      `WEBSOCKET_PORT=#port-number`

- Run `yarn install`
- Run `yarn remote-mouse:dev`
- The application will be up and running.
